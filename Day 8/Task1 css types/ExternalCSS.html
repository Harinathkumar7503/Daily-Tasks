About CSS Types

they are three 3 ways to applyCSS to HTML elements

Inline CSS:

  • INLINE CSS is the way of apply CSS to Html Elemnts.
  • here we use style tag as a atribute .
EX:
<h1 style="color:red";> hello </h1>

INTERNAL CSS:

  • INTERNAL CSS is the way of apply CSS to Html Elemnts.
  • here we use style tag.
  • Between style tag we have to write CSS Propertys.
  • the style tag is place Between <head> tag
EX:
<head>
<Style>
body{    background-color:red;
                            color:blue:
                        }
</Style>
</head>

EXTERNAL CSS:

  • EXTERNAL CSS is the way of apply CSS to Html Elemnts.
  • here we create another file
  • Inside the onther file we have to write CSS Propertys.
    • the style tag no need & save this file with .css extention
  • link this file with HTML file by <link> tag
    • EX:
    <link type = "stylesheet" href="example.css >
