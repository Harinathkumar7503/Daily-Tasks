CSS Animation + Keyframes

Syntax

selector {
animation-name: name;
animation-duration: time;
animation-iteration-count: number;
animation-timing-function: ease | linear | etc;
animation-delay: time;
animation-direction: normal | reverse | alternate;
}

Keyframes Syntax

@keyframes animationName {
from { property: value; }
to { property: value; }
}

Types Of Animation

FadeIN | FadeOut | ZoomIn | ZoomOut | BounceIn | BounceOut

FadeIn

Fade In animation is used to make an element gradually appear on the screen.

FadeOut

Gradually makes an element disappear by decreasing its opacity from 1 to 0.

ZoomIn

Smoothly increases the size of an element to make it appear closer or larger.

ZoomOut

Smoothly decreases the size of an element to make it appear smaller or distant.

BounceIn

Makes an element enter with a bouncing effect, starting fast and slightly overshooting.

bounceOut

Makes an element exit with a bounce effect, shrinking or moving away before disappearing.
